# 🖥️ AMLES Workstation Deployment

Procédure complète d’installation et de configuration des postes de travail de l’entreprise AMLES.

## 📌 Objectif

Mettre en place un environnement professionnel comprenant :

* Installation en **dual boot** (Windows 7 & Arch Linux)
* Configuration réseau
* Installation et configuration des logiciels métiers
* Mise en place des comptes utilisateurs et groupes
* Configuration des périphériques (imprimante réseau)

---

## ⚙️ Contenu du projet

Ce dépôt contient :

* 📄 Guide détaillé d’installation en dual boot
* 🪟 Configuration complète de Windows
* 🐧 Configuration complète de Arch Linux
* 🖨️ Installation et paramétrage d’une imprimante réseau
* 👥 Gestion des utilisateurs et des groupes

---

## 🧩 Technologies & Systèmes

* Windows 7
* Arch Linux
* GRUB (bootloader)
* XFCE & LightDM
* CUPS (gestion d’impression)
* Netctl / dhcpcd (réseau)
* LibreOffice / Firefox / Bluefish
